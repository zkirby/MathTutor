About
Hey there! My name is Zachary, I'm the developer and maintainer of this here proof of concept website. Nothing is functional as of right now and only the Calc AB module page actually exists, all that has been done is the very basics of the front end but hopefully I'll be able to hash out a little bit more in the coming weeks. Essentially this website theoretically is going to exist as a hub of tutoring. It's going to have tutorials, step by step derivative calculators and anything and everything a tutor could need to aide them in a tutoring session. This idea is based around the popular and helpful www.pythontutor.com.

The website is broken down into 'modules' which each cover a specific subject in math. Each one is chalk full of tutoring resources to make each tutoring session the most effective it can be. To get started search for a module or scroll down to see trending ones.
